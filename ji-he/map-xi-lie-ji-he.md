# Map系列集合

### Map 集合概述和使用

* Map集合是一种双列集合，每个元素包含两个数据
* Map集合的每个元素的格式： key = value (键值对元素)
* Map集合也被称为“键值对集合”

### Map 集合整体格式：

* Collection集合的格式:[元素1，元素2，元素3....]
* Map集合完整格式：{key1=value1,key2=value2,key3=value3,...}
